<async> joinGame(index)

Attempts to join the game at the specified index in the results.

If it fails to join the game for any reason, it will automatically retry with the next result. If it has reached the end of the results list, it will kick off another search and repeat from the start of those results, until it successfully joins a game or the user navigates away from the page.
