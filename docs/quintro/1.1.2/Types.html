Type Definitions

BoardPosition

Represents a position on a board.

This is a two-element array of positive integers, where the first integer is the column offset of the position (horizontal axis) and the second integer is the row offset of the position (vertical axis). Higher values for the column offset are closer to the right side of the board. Higher values for the row offset are closer to the bottom side of the board.

Type:
  • Array.<number>

BoardPositionImmutable

Represents a position on a board. Same as Types.BoardPosition, but represented as a two-element Immutable indexed collection, such as a external:Immutable.List.

Type:
  • external:Immutable.Collection.Indexed.<number>
