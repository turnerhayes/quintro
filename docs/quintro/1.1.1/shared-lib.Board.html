the amount and direction the column index should change with each step in the board traversal. Can be -1, 0, or 1.
rowDelta number

the amount and direction the row index should change with each step in the board traversal. Can be -1, 0, or 1. For example, to check the horizontal axis, columnDelta should be 1 and rowDelta should be 0.
